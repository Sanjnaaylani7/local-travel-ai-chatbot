# Web Widget Documentation

This README file provides an overview of the web widget component of the Local Travel AI Chatbot project. The web widget is designed to facilitate user interaction with the chatbot, allowing users to ask questions and receive responses related to travel information.

## Overview

The web widget is implemented in two versions: a simple HTML/JavaScript version and a React-based version. Both versions are designed to be easily integrated into existing websites.

### Features

- **User-Friendly Interface**: The widget provides a clean and intuitive interface for users to interact with the chatbot.
- **Multilingual Support**: The widget supports queries in English, Urdu, and Roman Urdu.
- **Responsive Design**: The widget is designed to work seamlessly on both desktop and mobile devices.
- **Fallback Options**: If the chatbot cannot provide an answer, users are directed to contact support for further assistance.

## Installation

### HTML/JavaScript Widget

1. Include the `widget.js` and `styles.css` files in your HTML.
2. Add the `index.html` file to your website to display the widget.

### React Widget

1. Navigate to the `react-widget` directory.
2. Install dependencies using npm or yarn:
   ```
   npm install
   ```
   or
   ```
   yarn install
   ```
3. Import the `ChatWidget` component into your application and include it in your JSX.

## Usage

Once integrated, users can click on the chat icon to open the chat panel. They can type their queries, and the chatbot will respond based on the local knowledge base.
